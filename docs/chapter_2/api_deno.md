## Deno API

window对象中存在一个Deno属性。
在Deno环境下，同过以下代码可查看其下的API。

```javascript
let apiArr = Object.keys(window.Deno);
console.log(apiArr.length);
```

```javascript
apiArr.forEach((api)=>{console.log(api+'\t')})
```
